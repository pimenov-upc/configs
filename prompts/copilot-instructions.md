# Загальні правила

- Завжди відповідай українською мовою.
- Використовуй 2 пробіли для відступів.
- Додавай JSDoc коментарі до кожної функції.
- Мій основний стек: TypeScript, JavaScript, HTML, CSS, SCSS.
- Основні бібліотеки: Vue 3, Vuetify, Pinia, vue-i18n, Axios.
- Інструменти: Vite, ESLint, Prettier, Vitest, oxlint, stylelint.
- Проекти збираються за допомогою Vite, тому оптимізація конфігурації Vite є важливою для покращення продуктивності.
- Завжди перевіряй критичні файли на витік на зовнішні ресурси, особливо якщо вони містять конфігурації або ключі, такі файли мають бути прописані в .gitignore та .npmignore (приклади фалів: .env, \*.pem, \*.crt, \*.key).
- Уникай використання `console.log` та `debugger` у коді, особливо у продакшн-версіях, щоб не залишати слідів відладки в кінцевому коді.
- Пам'ятай про важливість оптимізації залежностей та використання сучасних можливостей Vite для покращення швидкості збірки та продуктивності додатка.
